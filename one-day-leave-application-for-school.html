One Day Leave Application for School

Sometimes students need to take leave from school for personal reasons, illness, or family work. In such cases, students must inform the school through a proper leave application addressed to the principal or class teacher.

Below is a simple one day leave application format for school students. The application format is available in both English and Hindi so that students can easily copy and submit it.

How to Write One Day Leave Application for School

Write the application to the principal or class teacher. Mention the reason for taking leave. Clearly state that the leave is for one day. Write the date of leave. End the application politely and mention your name, class and roll number.

One Day Leave Application for School (English)

To The Principal [School Name] Date: ___ / ___ / 20__ Subject: Application for One Day Leave Respected Sir/Madam, I am a student of class ______ in your school. Due to personal reasons, I will not be able to attend school tomorrow. Therefore, I kindly request you to grant me leave for one day. Thank you. Yours sincerely Name: Class: Roll Number:

FAQ

When should a student write a one day leave application? Students should write this application when they need to take leave from school for a single day due to personal reasons or illness. Who should the application be addressed to? The application should be addressed to the school principal or class teacher. Is it necessary to write a leave application for one day? Yes, writing a leave application informs the school authorities about the student's absence.
