Leave Application for Exam Preparation

Students often write a leave application for exam preparation when upcoming examinations require extra study time. Informing the school or college through a proper leave application helps maintain attendance records and ensures permission from the principal or class teacher.

Below is a simple leave application format that students can use when requesting leave for exam preparation. The format is available in both English and Hindi so that students can easily copy and submit the application.

How to Write Leave Application for Exam Preparation

Mention the principal or class teacher. Write the subject clearly stating leave for exam preparation. Introduce yourself with your name and class. Explain that you need leave to prepare for upcoming exams. Mention the number of leave days. Request approval politely and end the application respectfully.

Example Leave Application for Exam Preparation

Below is a simple example of a leave application that students can submit to the school principal or class teacher when they need time to prepare for upcoming examinations. This format can be used by both school and college students.

Leave Application for Exam Preparation (English)

To The Principal [School / College Name] Date: ___ / ___ / 20__ Subject: Leave Application for Exam Preparation Respected Sir/Madam, I am a student of class ______ in your institution. Our examinations are approaching and I would like to take leave for a few days in order to focus on my exam preparation. Therefore, I kindly request you to grant me leave from ___ to ___ so that I can prepare properly for my upcoming examinations. I shall be grateful for your consideration. Thank you. Yours sincerely Name: Class: Roll No:
